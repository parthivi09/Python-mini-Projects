🐍 Python Playground: Console Games & Mini-Projects

Welcome to my Python learning journey! This repository is an organized collection of interactive, terminal-based games and utility applications.

Rather than creating separate repositories for every small project, I have organized them into dedicated subfolders. Each subfolder contains its own self-contained game script and an explanatory README.md file detailing how it works.

🛠️ Core Concepts Practiced

Through these projects, I am refining my understanding of:

Control Flow: while loops, infinite loops, break, continue, and nested if-elif-else conditions.

Functions: Defining clear responsibilities, passing parameters, and using return statements.

Data Structures: Dictionaries for input-to-value mapping and translation.

State Management: Tracking scores, lives, or balances across dynamic gameplay loop iterations.

🚀 How to Navigate and Play

Clone the entire repository:

git clone [https://github.com/YOUR_GITHUB_USERNAME/mini-python-projects.git](https://github.com/YOUR_GITHUB_USERNAME/mini-python-projects.git)
cd mini-python-projects


Navigate to a specific game folder and read its local README.md for specific rules:

cd rock_paper_scissors
python rock_paper_scissors.py
